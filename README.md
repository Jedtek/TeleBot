### Installation / Requirements

(Make sure you have typescript tsc in PATH/as a command)
```
pnpm install 
```

### Config

Make sure to copy config.sample.json to config.json and fill in the fields (currently all required for now)

### Running

To build:
```
pnpm run build
```

To start: 
```
pnpm run start
```

To build & start:
```
pnpm run buildStart
```
